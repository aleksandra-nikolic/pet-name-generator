# Pet Name Generator

A fun web application for generating pet names, built with Angular.
Filter names by gender and style, save your favorites and manage your list.

## Features
- Choose your pet type on the home page
- Filter names by gender and style (cute, wild, balkan...)
- Random name generator based on selected filters
- Restart option to generate a new random name
- Save favorite names to a list
- Delete names from favorites list
- Favorites saved with Local Storage

## Technologies
- Angular
- TypeScript
- HTML & CSS

## Run locally
1. Clone the repository
   git clone https://github.com/aleksandranikolic96/My-pet-generator.git
2. Install dependencies
   npm install
3. Start the development server
   ng serve
4. Open http://localhost:4200 in your browser
